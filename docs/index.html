核心能力

为开发者构建的 AI 助手

不只是聊天，而是真正能操作你的代码库的智能 Agent。

🔄

ReAct Agent 循环

自主思考 → 调用工具 → 观察结果 → 继续推理，直到完成任务。支持多轮工具调用和死循环检测。

📁

完整文件操作

读取、创建、精确编辑文件，支持带行号输出、大文件截断、字符串替换 patch，绝不破坏无关代码。

🔍

代码搜索

基于 ripgrep 的高性能正则搜索，以及 glob 文件模式匹配，快速定位代码位置。

🐚

Shell 执行

流式输出，50ms 节流渲染，支持 bash/zsh/powershell。运行测试、git 操作、构建命令一气呵成。

🤖

Sub-Agent 并行

通过 task 工具拆解复杂任务，启动子 Agent 并行执行，token 消耗自动汇总到主会话。

🔌

MCP 协议扩展

支持 stdio 和 HTTP 两种 MCP 传输方式，接入任意 MCP 服务器，无限扩展工具能力。

💾

会话持久化

对话自动以 JSONL 格式保存，支持跨 session 恢复。上下文超限时自动 LLM 压缩摘要，保持连续性。

⚡

Prompt Caching

Anthropic 自动注入最多 4 个 cache_control 断点，OpenAI 设置 promptCacheKey，系统提示字节稳定保证缓存命中。

🧠

快速开始

一分钟上手

全局安装后，在任意项目目录启动即可。

① 安装

npm install -g @mini-code-cli/cli

② 配置 API Key（以 Anthropic 为例）

export ANTHROPIC_API_KEY=sk-ant-... # 或写入项目 .env 文件，自动加载

③ 启动

mc # 短命令 mini-code # 完整命令

④ 常用示例

# 指定模型 mc --model deepseek # 信任模式（跳过确认） mc --trust # 计划模式（只读分析） mc --plan # 非交互单次输出 mc -p "解释这个函数的作用"

工作模式

三种权限模式

根据场景选择合适的安全策略。

default

默认模式

写操作（编辑文件、执行 Shell）前弹出权限确认。支持一次性允许、本 session 始终允许、拒绝三种选择。

--trust

信任模式

跳过所有权限确认，Agent 可以自由执行所有操作。适合在受控 CI 环境或你完全信任的场景中使用。

--plan

计划模式

只读模式，Agent 只能读取文件和分析代码。先让 AI 制定方案，确认后再切换到普通模式执行。

AGENTS.md 5 层上下文

向 Agent 注入项目规范和个人偏好，优先级从低到高自动合并。

1

用户全局规则

~/.mini-code/AGENTS.md

2

用户自动记忆

~/.mini-code/auto-memory.md

Agent 自动提取并记录的用户偏好

3

项目规则链

<项目路径>/AGENTS.md

从 git 根目录到当前目录，越具体优先级越高

4

项目自动记忆

<cwd>/.mini-code/auto-memory.md

Agent 自动记录的项目级事实

5

本地私人偏好（最高优先级）

<cwd>/AGENTS.local.md

个人专属配置，建议加入 .gitignore 不提交
